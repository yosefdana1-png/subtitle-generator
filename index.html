כתובית

מעלים סרטון, מקבלים כתוביות בעברית מסונכרנות אוטומטית, ומורידים קובץ SRT. התמלול רץ ישירות מהדפדפן שלך אל OpenAI — שום קובץ לא נשמר בשום מקום אחר.

מפתח API של OpenAI

המפתח נשמר בדפדפן הזה בלבד (על המכשיר שלכם), ולא מועבר לשום מקום חוץ מ-OpenAI. אפשר ליצור מפתח בכתובת platform.openai.com/api-keys

כתובת שרת מתווך (Worker)

OpenAI חוסמת קריאות ישירות מהדפדפן, אז צריך שרת קטנטן שרק מעביר את הבקשה הלאה. הכתובת שתקבל אחרי הקמת ה-Worker נכנסת כאן.

הקשר לתמלול (אופציונלי)

מונחים, שמות או ז'רגון שחוזרים בסרטון — עוזר ל-Whisper לזהות אותם נכון במקום לנחש מילה דומה. לא חובה.
